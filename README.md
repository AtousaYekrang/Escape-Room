# Escape-Room Game

## Description
An interactive web-based escape room game developed as a university team project.

## Features
✅ Interactive puzzle solving
✅ Game logic and user interactions
✅ Responsive UI design
✅ JavaScript game mechanics

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## How to Play
1. Open `index.html` in your browser
2. Solve the puzzles to escape
3. Click on objects to interact with the game

## Installation
```bash
git clone https://github.com/AtousaYekrang/Escape-Room.git
cd Escape-Room
open index.html
```

Developed as a university collaborative project.
